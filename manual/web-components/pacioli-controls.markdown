# Pacioli Controls

The `pacioli-controls` component controls animations.

## pacioli-value

The `pacioli-controls` component adds a panel to controls animations.

    <pacioli-controls for="..."></pacioli-controls>

The value of the `for` atrribute is the id of the scene to control.

Use a comma-separated list of scene ids to control multiple scenes
with one control panel.

## Common Attributes

No common attributes.

## Pacioli Controls Attributes

<dl>
  <dt>for</dt>
  <dd>Scene id or comma-separated list of scene ids</dd>
</dl>
