# Library uom-ts

A typescript library for units of measurement. Part of [Pacioli][1].

Uses [bignumber.js][2] in the implementation of dimensioned numbers.

# Build the library

Update the version in `package.json` if necessary and run

```
npm run package
```

This generates a .tgz file.

The JavaScript code is build with esbuild. The types are generated by tsc. Use `npm run package:tsc` to
compile the code with tsc instead of esbuild. Clear the dist directory when switching between the esbuild
and the tsc variant to avoid mixing the output.

# Testing the library

The tests use [Vitest][3] and [fast-check][4].

## Running the tests

To run the tests in 'watch' mode:

```
npm run test
```

Run an extended test with:

```
npm run test:longruns
```

See the Vitest manual for info about debugging the tests, setting breakpoint, etc.

A failing test can be rerun by pasting the { seed: 1442240156, path: "149448", endOnFailure: true } as second arg of the fc.assert of the failing test.

## Test configuration

See the vitest.config and vitest.setup files.
