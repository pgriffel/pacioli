# Parametric Surfaces

Some examples of parametric surfaces.

A surface is described parametrically with a function $f(u,v)=(x, y, z)^T$
to a three-dimensional position vector. The domains of $u$ and $v$ vary per parametrization.

The complete code for the examples can be found [here](https://raw.githubusercontent.com/pgriffel/pacioli/develop/samples/parametric_surfaces/surfaces.pacioli)
