Parametrization examples

These examples show parametrically rendered two-dimensional surfaces.

A surface is described parametrically with a function from parameters (u, v) to a three-dimensional position vector. The domains of u and v vary per parametrization.

Parameter n is the number of samples in each of the two directions. The computation complexity is therefore n squared. Parameter m is the density of the normal and other vectors. If m equals 1 then arrows are shown for every line segment. If m equals 2 then are shown every other segment. For m equals 3 every third segment. And so on. Reducing the number of arrows helps performance. Parameter d is a scaling factor for the arrows. Increasing d increases the arrow sizes.
