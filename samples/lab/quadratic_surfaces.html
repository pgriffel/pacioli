Quadratic and paraboloid examples

These examples render two-dimensional surfaces.

First we express a class of quadratic forms that can be used to render either a paraboloid or a saddle surface.

Next, for comparison, we show the parametrically defined paraboloid function.

In both cases we observe that the z axis has a metre squared as its unit.

A surface is described parametrically with a function f ( u , v ) = ( x , y , z ) T from parameters (u, v) to a three-dimensional position vector. The domains of u and v vary per parametrization.

In the examples below, parameter n is the number of samples in each of the two directions. The computation complexity is therefore n squared.

Parameter m is the density of the normal and other vectors. If m equals 1 then arrows are shown for every line segment. If m equals 2 then are shown every other segment, and so on. Reducing the number of arrows helps performance.

Parameter d is a scaling factor for the arrows. Increasing d increases the arrow sizes.

Paraboloid and saddle surface as quadratic form

We consider the class of quadratic forms that is expressed as:

z = ( x y ) ( A x x A x y A y x A y y ) ( x y )

( x y ) in this case is computed as the "normal" transpose operator (as opposed to the reciprocal or dimensional inverse).

We let the variables x and y be parametrically defined as follows:

  • x = a u cos (v)
  • y = a u sin (v)
where a is a scaling factor, 0 ≤ u ≤ 5 , and 0 ≤ v ≤ 2π

As the quadratic form has both x and y squared, z 's unit is metre squared.
