# Isometry definition as in Roman, Advanced Linear Algebra
#
# Isometry is defined through a normed linear space (p209);
# - the norm(v) = length(v) = sqrt(inner(v, v))
# - the distance between two vectors u and v given by d(u,v) = norm(u-v)
# as in the functions metric_norm and metric_distance aboven.
# We speak of "norm" only When the distance properties match that of a metric space;
# e.g. d(u,v) = 0 <=> u=v.
#
# In this approach the Unitary (complex) and Orthogonal (real) operators represent
# the isometries, following 10.12 on p240. The adoint equals the inverse T* = T^-1
# These operators als form special types of normal operators - they commute with their
# own adjoint TT* = T*T. 
#
# We work out the Orthogonal (real) case below.
#
