The petri_net library

The Petri net library provides a matrix representation of dimensioned Petri nets. In a dimensioned Petri net a unit of measurement is associated with the net's places.

A Petri net is a tuple ( P , T , F + , F − )

with

  • P a set of places,
  • T a set of transitions,
  • F + a matrix of the form P × T → ℕ called the post matrix, and
  • F − a matrix of the form P × T → ℕ called the pre matrix.

A Petri net's state is defined by its marking. A marking m is a vector P → ℕ .

For any place p ∈ P value m p is the number of tokens in that place.

A Petri net changes state by firing transitions. Given a marking m , firing a transition t ∈ T is enabled when F p t − ≥ m p for each p ∈ P . When the transition fires it consumes F p t − tokens from each place, and produces F p t + tokens for each place.

In a matrix setting it is natural to define firing of transitions in parallel. Given a transition vector x of type T → ℕ where x t is number of times transition t ∈ T fires, the new marking m ' is given by

m ' = m + F x under the condition that F − x ≥ m

where flow matrix F , is defined as F = F + - F −
