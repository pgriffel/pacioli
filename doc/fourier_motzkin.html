fourier_motzkin

:: for_index P, Q: for_unit a, P!u: (a*P!u per Q!) -> List(Q!)

All nullifiers of the matrix. That is, all integer vectors v such that matrix '*' v is zero.

The matrix must be an integer matrix.

The algorithm is exponential in the row dimension and in the column dimension.
